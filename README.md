# StrataNode Infrastructure Ltd

## Connecting Ireland's Low-Altitude Economy

A public industry discussion paper exploring how Ireland could originate, assess and develop the physical, regulatory, digital and commercial infrastructure required for future low-altitude operations.

### Key themes

- Corridor origination
- Infrastructure nodes
- Planning and compliance
- Digital infrastructure
- Connectivity
- Commercialisation
- Multi-operator interoperability

Review it. Challenge it. Improve it.
